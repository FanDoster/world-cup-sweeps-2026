# World Cup 2026 Sweepstakes

Live: [world-cup-sweeps-2026.surge.sh](https://world-cup-sweeps-2026.surge.sh)

Six-player sweepstakes tracker for the 2026 FIFA World Cup. Each player drew 8 teams. Tracks group stage matches, results, and a live leaderboard.

## Features
- Match schedule with countdown timers and UK TV channels (BBC/ITV)
- Win/draw/loss probability bars (Elo-based from FIFA rankings)
- Group stage tables with team ownership
- Live leaderboard with auto-calculated points (3pts win, 1pt draw)
- Team schedule view — click any team chip to see their fixtures
- Mobile responsive with touch-friendly probability display

## Deploy
Single static HTML file. Auto-deploys to Surge on push via GitHub Actions.
